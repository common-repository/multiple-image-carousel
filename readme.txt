== Description ==
The WordPress Plugin “Multiple Image Carousel ” is the excellent choice for adding multiple image carousel on your website. It consists of several great modules which are easy to manage from the admin dashboard of the website.

In addition, this plugin has the features to generate shortcode to display the carousel on single or multiple pages of your site. 

Multiple Image Carousel Plugin has so many features which are listed below. You can take advantage of this plugin easily and free of cost from WordPress plugindirectory. You can enable or disable the option for autoplay image carousel. 


Features:

* Auto play Enable/Disable option.

*  Number of Items Show option.

* Loop Enable/Disable option.

* Enable/Disable Navigation option.

* Enable/Disable Dots option.

* Auto play Stop On Hover option.

* Center Mode Enable/Disable option.

* change Direction option.

* Enable/Disable Touch Drag option.

* Enable/Disable Mouse Drag option.

* set Margin option.

* set Stage Padding option.

* Slide By option 

* Start Position option 

* Autoplay Speed option

* Fully  Responsive.

* Easy Shortcode Generator.

* Unlimited Number of Carousel Items.

* All Browser support.




== Installation ==
Installing "Multiple Image Carousel" can be done either by searching for "Multiple Image Carousel" via the "Plugins > Add New" screen in your WordPress dashboard, or 

by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
