== Description ==
This is an excellent plugin for Adding multiple image carousel.

It has wonderful modules that are easy to manage the admin section of the website. In Addition to this, it automatically generates shortcode and you can show multiple image carousel on a single page or multiple pages.


Features:

* Auto play Enable/Disable option.

*  Number of Items Show option.

* Loop Enable/Disable option.

* Enable/Disable Navigation option.

* Enable/Disable Dots option.

* Auto play Stop On Hover option.

* Center Mode Enable/Disable option.

* change Direction option.

* Enable/Disable Touch Drag option.

* Enable/Disable Mouse Drag option.

* set Margin option.

* set Stage Padding option.

* Slide By option 

* Start Position option 

* Autoplay Speed option

* Fully  Responsive.

* Easy Shortcode Generator.

* Unlimited Number of Carousel Items.

* All Browser support.




== Installation ==
Installing "Multiple Image Carousel" can be done either by searching for "Multiple Image Carousel" via the "Plugins > Add New" screen in your WordPress dashboard, or 

by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
